#Tushare, the library used to fetch financial data

[Tushare](http://tushare.org/)

*Tushare is a free, open sourced python financial data interface library*.
Its primary features include collecting data of financial, such as stock; cleaning data and storing data,
which provides financial analysts with quick, neat, and diversy data for analysis, reduces difficulty on 
data collection for them, allows them to focus more on realization of strategy and modeling....

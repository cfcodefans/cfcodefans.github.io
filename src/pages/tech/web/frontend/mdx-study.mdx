Let us try making a react component in mdx file.

```javascript
function TemperatureConvertor(): JSX.Element {
    const inputRef = useRef(null)
    const [ts, setTS] = useState(0)

    return <>
        <input type="number" 
            ref={inputRef} 
            onChange={ev => setTS(inputRef.current.value)} 
            value={ts}></input> 째C =&gt; 
        <span>&nbsp;{ts * 9 / 5 + 32} 째F</span>
    </>
}
```

However, **directly coding in mdx file**, doesn't take the componet to live on the page.
It has to be code somewhere in .jsx or .tsx file, then import into the mdx file.
The import clause has to be placed on top of the mdx file.
